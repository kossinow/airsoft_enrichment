Поделка для ролевой страйкбольной игры.

Установка для обогащения капсул-снарядов.

Состоит из:
  Arduino nano
  Дисплей LCD1602
  Геркон
  Транзистор
  Кнопка нормально разомкнутая

Описание работы устройства:
При подключении питания на экране отобразится надпись "Установи капсулу",
после того как она будет установлена, начнется обратный отсчет.
После завершения отсчета капсула будет засвечена в течение 3 секунд,
прозвучит звуковой сигнал и отобразится надпись "Капсула активирована".
После того как капсула будет извлечена, устройство перейдет в исходное состояние.
Если извлечь капсулу до завершения таймера, время будет сброшено.

Настройка устройства:
Для перехода в режим изменения таймера необходимо подать питание при зажатой кнопке.
Кнопку можно отпустить когда на экране появится надпись "Установи время",
однократным нажатием кнопки можно выбрать подходящий таймер,
после чего просто отключить питание. 
При последующих включениях устройство отработает с выбранным таймером.
